Projects

My Company Page

In this project, I used Flexbox to design and build the layout for my company's homepage. I designed the content for this website including, title, logo, mission statement, images and description of the company. Flexbox styling was used for the layout including horizontal and vertical positioning, flexible element flows as the page size changes, and great styling for repeated elements. I used custom styles for other properties, such as colors, fonts, and other layout properties such as borders and padding.

Kenzie's Park

In this project, I built a responsive website using @media rules to change page layout and behavior based on size breakpoints. I created all elements relating to this page, including the name, page layout, styling and the images. I demonstrate many of the responsive design tools I've learned, such as percentages and relative units to size and space elements with width and padding. I also incorporated media queries to resize elements based on the size of the screen, making images toggle to certain percentages in terms of width after a certain minimum screen size.

Number Guesser

In this project, I wrote four JavaScript functions to power a small guessing game, which were incorporated into a website that also uses HTML/CSS. The functions included generating a random integer between 0 and 9, to be called at the start of each new round to generate the new secret target number. Another function to be called each round to determine which guess is closest to the target number. A third function to increase the winner's score after each round, with the final function updating the round number after each round.
